# 01 游戏设计文档：核心玩法 GDD

## 1. 游戏一句话描述

一款横屏 2D 放置合成塔防游戏。玩家在固定塔位放置中世纪奇幻英雄，通过商店购买和三合一升星构筑阵容，阻挡一波波沿轨道进攻的怪物，保护 10 点生命的城门核心。

## 2. 玩家目标

单关目标：在生命归零前守住 10 波敌人。

长期目标：解锁更高难度、更多地图、更多队长英雄、更多职业组合和更高分数。

## 3. 核心循环

### 3.1 局内循环

```text
准备阶段
  -> 查看下一波预告
  -> 购买英雄
  -> 刷新商店
  -> 三合一升星
  -> 放置或换位
  -> 开始下一波
战斗阶段
  -> 敌人沿路径出现
  -> 英雄自动攻击
  -> 近战英雄阻挡
  -> 玩家释放队长技能/法术
  -> 敌人死亡或漏怪
结算阶段
  -> 清波奖励
  -> 完美防守奖励
  -> 队长经验
  -> 返回准备阶段
```

### 3.2 关卡循环

```text
第 1 波教学 -> 第 4 波出现飞行 -> 第 6 波出现重甲 -> 第 7 波出现潜行或辅助 -> 第 10 波 Boss
```

### 3.3 成长循环

```text
买 1 星 -> 三个 1 星合成 2 星 -> 解锁职业核心技能 -> 三个 2 星合成 3 星 -> 解锁终极强化
```

## 4. 胜负规则

| 条件 | 结果 |
|---|---|
| 第 10 波所有敌人被消灭且生命 > 0 | 胜利 |
| 生命降到 0 | 失败 |
| Boss 漏怪但生命仍 > 0 | 继续，扣 Boss 漏怪血 |
| 玩家退出 | 保存当前关卡进度或返回主菜单 |

## 5. 生命规则

每关 10 点生命。漏怪扣血由敌人决定：

| 体型 | 扣血 |
|---|---:|
| 小型 | 1 |
| 中型 | 1-2 |
| 精英/辅助 | 2 |
| Boss | 3 |

生命提示：

- 10-8：安全，绿色。
- 7-4：警戒，黄色。
- 3-1：危险，红色，屏幕轻微闪烁。
- 0：失败。

## 6. 经济规则

### 6.1 金币来源

| 来源 | 规则 |
|---|---|
| 清波奖励 | `4 + floor(waveIndex / 2)` |
| 击杀奖励 | 按敌人类型给 1-6 金币 |
| 完美防守 | 本波未漏怪给 +2 |
| Boss 击杀 | 额外 +8 到 +16 |

### 6.2 金币用途

| 用途 | 价格 |
|---|---:|
| 购买 1 星英雄 | 3-6 |
| 刷新商店 | 第 1/2 次 2，之后 3-4 |
| 出售英雄 | 返还基础价格 50% |
| 塔位强化 | Alpha 后加入 |

## 7. 商店规则

- 商店 3 格。
- 每波准备阶段自动刷新。
- 玩家可花金币刷新。
- 购买失败不移除商店项。
- 商店权重根据波次调整：前期更容易出弓箭手、战士、骑士；中后期提高法师、刺客、牧师。
- 商店只出售守卫英雄，不出售队长。

## 8. 三合一规则

| 合成 | 结果 |
|---|---|
| 3 个同职业 1 星 | 1 个同职业 2 星 |
| 3 个同职业 2 星 | 1 个同职业 3 星 |
| 3 星重复 | 转星尘或金币返还，MVP 可先忽略 |

细节：

- 合成只在准备阶段自动触发。
- 塔位上英雄可参与合成，但优先保留塔位上的最高击杀实例。
- 队长英雄不参与合成。
- 锁定英雄不参与合成。
- 合成后继承最高击杀、最高经验标记、塔位位置。

## 9. 放置规则

| 职业 | 放置方式 |
|---|---|
| 弓箭手 | 塔位固定炮台 |
| 法师 | 塔位固定炮台 |
| 牧师 | 塔位固定辅助 |
| 战士 | 塔位附近驻守，阻挡地面敌人 |
| 骑士 | 塔位附近驻守，高阻挡 |
| 刺客 | 塔位出击，突袭高价值目标后返回 |

## 10. 目标选择规则

默认目标优先级：

1. 最接近终点的敌人。
2. 飞行单位若英雄对空且飞行优先。
3. Boss 如果进入范围，部分英雄优先 Boss。
4. 刺客优先辅助/远程/术士。
5. 牧师优先治疗血量百分比最低的友方。

可选目标策略后续加入：

- First：最靠前。
- Last：最后。
- Strong：血量最高。
- Fast：速度最快。
- Air：飞行优先。
- Support：辅助优先。

## 11. 难度

| 难度 | 生命 | 金币 | 敌人 |
|---|---:|---:|---|
| 新手 | 10 | +15% | 生命 -10%，速度 -5% |
| 普通 | 10 | 标准 | 标准 |
| 困难 | 8 | -10% | 生命 +15%，精英更多 |
| 无尽 | 10 | 标准 | 每 5 波强化 |

MVP 只做普通。

## 12. 首次可玩体验

第一关前 3 波必须让玩家学会：

1. 放置弓箭手。
2. 点击开始下一波。
3. 看到敌人沿路走。
4. 看到英雄攻击。
5. 买到第三个相同英雄触发合成。
6. 第 4 波看到飞行怪，理解对空。

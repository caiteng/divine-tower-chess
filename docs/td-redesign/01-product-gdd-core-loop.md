# 01 产品 GDD：核心玩法、经济、合成、胜负

## 1. 游戏一句话

一款中世纪奇幻风格的放置合成塔防：玩家购买并合成英雄，把英雄布置在地图塔位上阻挡和消灭沿路径进攻的敌人，使用队长技能扭转关键波次。

## 2. 核心循环

```text
选关 -> 选队长 -> 准备阶段 -> 商店购买 -> 放置/换位 -> 三合一升星 -> 开波 -> 自动战斗 -> 主动技能 -> 清波结算 -> 下一波
```

## 3. 单局结构

每关固定 10 波。

| 阶段 | 玩家行为 | 系统行为 |
|---|---|---|
| 准备阶段 | 买英雄、放置、合成、出售、刷新 | 商店展示 3 格，地图展示塔位 |
| 出怪阶段 | 观察压力、释放技能 | 敌人按队列生成 |
| 战斗阶段 | 使用队长技能、全局法术 | 英雄自动索敌攻击，敌人推进 |
| 结算阶段 | 领取金币、调整阵容 | 发放奖励、刷新商店 |

## 4. 生命规则

- 每关 10 点生命。
- 小怪漏过扣 1。
- 中型精英漏过扣 2。
- Boss 漏过扣 3。
- 生命为 0 立即失败。
- 清完第 10 波且生命大于 0 胜利。

## 5. 经济规则

| 资源 | 来源 | 用途 |
|---|---|---|
| 金币 | 波次奖励、击杀奖励、完美奖励 | 买英雄、刷新商店、临时强化 |
| 队长经验 | 清波、精英、Boss | 队长升级 |
| 星尘 | 售出、3 星溢出 | 法术、重置、后期系统 |

MVP 只实现金币。推荐起始金币 20；第 1 波奖励 5；后续基础奖励 `4 + floor(波次 / 2)`；刷新商店成本 `2 / 2 / 3 / 4+`。

## 6. 商店规则

- 商店 3 格。
- 每个准备阶段可刷新。
- 金币不足购买失败。
- 背包满购买失败。
- 购买失败不能吞商店格。
- 买到的英雄都是 1 星。
- MVP 英雄池：弓箭手、法师、战士、骑士、刺客、牧师。

## 7. 三合一规则

- 3 个同职业 1 星 -> 1 个同职业 2 星。
- 3 个同职业 2 星 -> 1 个同职业 3 星。
- 3 星封顶。
- 队长不参与合成。
- 锁定单位不参与合成。
- 合成优先消耗背包单位，再消耗塔位单位。
- 合成保留最高击杀数和最高强化等级。
- 合成发生在准备阶段，MVP 不在战斗中自动合成。

## 8. 放置规则

- 只有准备阶段能放置。
- 每个塔位最多 1 名守卫英雄。
- 远程职业固定在塔位。
- 近战职业以塔位为中心形成驻守圈。
- 飞行敌人不能被无对空能力的单位攻击。
- 点击背包英雄，再点击塔位完成放置。

## 9. 战斗规则

默认索敌：路径进度最高的可攻击目标。MVP 只实现最前。后续可扩展：最后、血量最高、飞行优先、精英优先。

伤害公式：

```text
原始伤害 = 攻击力 * 星级系数 * 技能系数
护甲修正 = max(1, 原始伤害 - 护甲 * (1 - 穿甲率))
最终伤害 = round(护甲修正 * 属性修正)
```

星级系数：1 星 1.00，2 星 1.72，3 星 2.95。

## 10. 胜负和奖励

胜利：第 10 波全部敌人死亡或离场，并且生命 > 0。

失败：生命 <= 0。

完美奖励：通关生命 >= 8 额外金币 +5；通关生命 = 10 后续可给星尘 +1。

# 16 阶段 9：五关 Alpha 内容

## 阶段目标

从单关 MVP 扩展到五关 Alpha。实现 5 张地图、50 波、8 类敌人、完整基础职业可玩。


## 每阶段固定要求

每个阶段提交必须包含：

1. 代码或文档变更清单。
2. 自动化测试。
3. 人工验收步骤。
4. 回退方案。
5. 下一阶段前置条件。
6. 不修改无关旧主线。

## 数值平衡目标

| 1 | 90% | 75% |
| 2 | 80% | 60% |
| 3 | 70% | 50% |

| 5 | 50% | 30% |

## 每关教学点

| 关卡 | 必须教会 |
|---|---|
| 1 | 放置、开波、合成、对空 |
| 2 | 多线补防 |
| 3 | 重甲和魔法解法 |
| 4 | AOE、阻挡、法术时机 |
| 5 | 完整阵容和 Boss 阶段 |

## 测试任务

- 每关 10 波配置完整。
- 每关至少 1 条地面路径。
- 每关飞行路径存在或明确没有飞行波。
- 每关塔位不重叠。
- 每关 Boss 波能生成。
- 所有敌人至少出现一次。
- 所有波次奖励为正。
- 每关能进入 victory/defeat。

## 验收

Cocos UI 可选择 5 关。每关都能开波、出怪、结束。

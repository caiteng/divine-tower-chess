# 游戏设计草案

## 项目类型
单机塔防原型游戏，结合自走棋式升星和神品进阶机制。

## 核心玩法
玩家每回合通过商店购买棋子，摆放在地图上的固定位置，抵御怪物沿路径进攻水晶。
击杀怪物可获得金币，用于购买棋子和刷新商店。
相同棋子可通过 3 合 1 合成为更高星级：
- 3个1星 -> 1个2星
- 3个2星 -> 1个3星

3星为普通最高星级。
当棋子达到3星后，每回合开始时有10%概率获得一个神品进阶任务。
当前版本仅实现两种神品进阶：
1. 战士 -> 狂战士：累计击杀1000个怪物后进阶
2. 牧师 -> 光法师：累计治疗100000生命后进阶

任务进度在同一局游戏内跨回合累积。

## 回合流程
1. 回合开始
2. 自动刷新商店，展示3个棋子
3. 玩家可购买棋子、刷新商店、摆放棋子、完成升星
4. 进入战斗阶段
5. 怪物沿路径进攻水晶
6. 棋子自动攻击与施法
7. 击杀怪物后获得金币
8. 清空当前波次后进入下一回合
9. 水晶生命归零则失败

## 难度
- 新手：10波
- 普通：30波
- 困难：60波

## 地图
第一版仅实现1张地图，带1到2条固定扭曲路线。

## 第一版棋子
- 弓箭手
- 圣骑士
- 盾卫
- 战士
- 法师
- 牧师
- 骑兵
- 枪兵

## 神品棋子
- 狂战士（由战士进阶）
- 光法师（由牧师进阶）

## 第一版目标
做出完整可玩的原型：
- 可选择难度
- 可招募棋子
- 可摆放棋子
- 可自动战斗
- 可升星
- 可触发并完成神品进阶任务
- 可完成一局游戏流程

## 非目标
- 不追求高质量美术
- 不追求完整平衡
- 不追求无bug
- 不实现联网、排行榜、广告、装备系统、复杂羁绊

# PLANS.md

## 当前状态（2026-04-27）

- 项目名为《角斗场：试炼之环》，主线是 Cocos Creator + TypeScript 可玩原型：Battleheart 风格 2D 小队实时命令战斗 + 商店 / 金币 / 3 合 1 升星 / 实例级神品任务。
- 正式入口流程：`主菜单 -> 职业 / 难度选择 -> 准备阶段 -> 战斗阶段`。
- 逻辑主链：
  - `assets/scripts/squad/squad-battle-session.ts`
  - `assets/scripts/squad/systems/*`
- Cocos UI 主链：
  - `assets/scripts/ui/battle-scene-controller.ts`
  - `assets/scripts/ui/squad-battle-ui.ts`
  - `assets/scripts/ui/controllers/*`
  - `assets/scripts/ui/views/*`
- 配置与资源主链：
  - `assets/scripts/config/unit-config.ts`
  - `assets/scripts/config/art-resource-manifest.ts`
  - `assets/scripts/config/unit-star-sprite-config.ts`
  - `assets/scripts/squad/config/squad-battle-config.ts`
  - `assets/resources/textures/**`

## 工程清洁规则

- 不入库目录：
  - `node_modules/`
  - `build/`
  - `dist/`
  - `library/`
  - `temp/`
  - `local/`
  - `tmp/`
  - `coverage/`
- `build/web-mobile/` 是 Cocos 构建产物，不作为源码管理。
- `tmp/web-e2e/` 是 Web e2e 截图、summary 和浏览器 profile 输出，不作为源码管理。
- `assets/**.meta` 是 Cocos 真实资源元数据，和对应资源一起入库。
- `tools/**` 是 Node/脚本验证工具，不放进 `assets/scripts/**`，避免被 Cocos 当作运行时代码打包。

## 手动验证步骤

1. 在 Cocos Creator 中打开项目并执行 Preview。
2. 从主菜单点击开始，进入职业 / 难度选择页面。
3. 在职业 / 难度选择页面分别选择新手、普通、困难、无尽，并左右切换职业，确认立绘完整且不会串图或被错误拉伸。
4. 确认战士职业预览显示完整站立图，不再是半身或裁切图。
5. 选择任意职业和难度开始游戏，进入准备阶段。
6. 确认准备阶段只显示备战页，不露出战场、HUD 或命令栏。
7. 确认商店显示 3 个单位，金币、备战区、上阵区显示正常。
8. 点击商店角色，确认角色正下方显示 `购买`，点击后成功加入备战区；金币不足或备战区满时商店槽位不消失。
9. 连续购买 3 个同名同星单位，确认自动合成 1 个 2 星单位且游戏不闪退。
10. 点击备战区角色，确认角色正下方显示 `上阵` 和 `售出`；点击上阵后进入上阵区。
11. 点击上阵区角色，确认角色正下方显示 `下阵`；点击后回到备战区。
12. 点击右侧显眼的 `开始下一波`，确认出现电影转场遮罩后进入战斗页，备战页隐藏。
13. 确认己方默认出生在地图下半区，不会站到背景天空区域。
14. 战斗中点击地图上半区或地图外，确认己方移动目标被限制在下半区作战带和地图边界内。
15. 让盾卫 / 战士身边已有敌人时观察其行为，确认会站定战斗 / 举盾，不会突然追远处敌人乱跑。
16. 让盾卫 / 战士无贴身敌人、附近有敌人时观察其行为，确认会小范围靠近战斗。
17. 战斗中点击己方单位，分别点地面、敌人、友军，确认移动 / 集火 / 治疗命令反馈正常。
18. 清完一波，确认出现电影转场遮罩后回到准备页，金币增加、商店自动刷新、战场和 HUD 隐藏。
19. 观察多个近战单位或敌人聚集时不会长时间完全重叠，会按体积互相推开。
20. 使用法师攻击聚集敌人，确认附近敌人也会掉血。
21. 使用猎人/法师攻击，确认有弹道、红橙色 PNG 伤害数字、命中闪光和脚底尘土。
22. 牧师治疗受伤友军，确认有治疗线、`+` 号绿色 PNG 治疗数字，并且满血目标不会产生治疗数字。
23. 击杀敌人，确认敌人最后位置播放死亡烟尘。
24. 进入 Boss 波，确认战场中央偏上显示 Boss 入场横幅。
25. 战斗中让盾卫移动、集火、承伤和倒下，确认分别播放 move、attack、idle_heavy / block、hurt、death 动作帧。
26. 战斗中让己方单位向左移动，确认贴图水平翻转后正向行走；再向右移动，确认恢复朝右。
27. 观察从左侧刷出的敌人或被推向右侧移动的敌人，确认敌人向右移动时会水平翻转，不会倒退行走。
28. 合成或载入 2 星猎人，战斗阶段选中猎人，确认底部出现 `狙击` 技能按钮。
29. 点击 `狙击`，确认技能自动锁定当前集火目标或最近敌人，产生弹道 / 伤害数字，并进入冷却倒计时。
30. 使用猎人或法师攻击敌人，在弹道出手后立刻点地面移动，确认弹道继续飞行 / 伤害正常结算，单位能马上开始移动。
31. 在猎人 / 法师刚开始抬手但弹道还没出现时点地面移动，确认这次攻击被取消，不产生弹道或伤害。
